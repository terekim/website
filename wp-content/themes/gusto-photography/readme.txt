== Description ==

Gusto Photography is a simple WordPress blogging theme focusing on images, use it for a travel, food or photo bloging. Gusto Photgraphy can also be used as a portfolio for creatives and photographers, tell your story using images.


== Frequently Asked Questions ==

= Does this theme support any plugins? =

gusto photography includes support for Infinite Scroll in Jetpack. 

= My Images are not the same size? =

if you had already uploaded images, you can use a plugin like 'thumbnail regenerate' to batch resize images

= I neeed help =

If you need support simple post on the publick forum on wordpress.org and we'll get back to you
